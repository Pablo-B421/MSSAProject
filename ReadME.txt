This Project is an all purpose baby activity tracker designed for use by first
time parents.
The tracker will be based around and event calendar that will be able to store
information such as:
	doctors appointments
		-babies weight
		-babies height
		-any other important information
	scheduled feeding times
	Milestones
	diaper changes.
This project is still under development.
This is an ongoing project that is used to apply the skills learned during the 
Microsoft Software and Systems Academy at Embry Riddle Aeronautical University
