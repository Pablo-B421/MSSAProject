BabyTracker
An all purpose calendar based tracker designed for new parents allowing them to track feeding times, diaper changes, doctors appointments and mile stones. The user will make a profile for the child, and input the child's name, date of birth, and gender(optional). 

Event Calendar
This is where all events such as diaper changes, feeding times, doctor appointments and where the user can track any releveant information

Feeding Tracker
Tracks the child's feeding times based on manual user input or time intervals chosen by the user. All changes and updates will be updated on the event calendar.

Diaper Tracker
Tracks the child's diaper changes based on manual user input. All changed and updates will be updated on the event calendar.

MileStones
MileStones will automatically be uploaded in the event calendar based on the child's date of birth
